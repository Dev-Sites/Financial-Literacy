What is metadata?

Metadata is descriptive information about a resource, sometimes described as 'data about data'. The vocational education and training (VET) sector's national metadata standard is called Vetadata. It is primarily used to create descriptive information for learning content so that it can be more easily found, managed and reused.

Examples of information commonly stored as a metadata record include authorship, title, publication date, description, copyright information, and subject keywords.

The Metadata for this object is contained in the 'metadata.xml' file, in the root level of the object.
